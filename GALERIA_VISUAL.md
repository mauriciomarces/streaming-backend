## 5. Animaciones y Efectos

```
HOVER EFFECTS:
  • Cards en dashboard: Eleva 8px + cambia borde a rojo
  • Tarjetas de historial: Mueve 4px a la derecha
  • Botones: Escala 1.05 + sombra roja
  • Users pills: Cambia color de borde a rojo

TRANSICIONES:
  • 0.2s - ease-in-out para transforms
  • 0.3s - smooth para colores
  • Barra de progreso: 0.5s para llenar

GRADIENTES:
  • Fondo: #0a0a0f → #1a0a0a (135deg)
  • Botones: #e50914 → #ff6b6b (135deg)
  • Header: negro 80% + rojo 5% (180deg)

SOMBRAS:
  • Cards hover: 0 20px 40px rgba(229,9,20,0.2)
  • Botones hover: 0 8px 20px rgba(229,9,20,0.4)
```

## 6. Responsive Breakpoints

```
Desktop (>1024px):
  • Dashboard: 4 columnas de stats
  • Users grid: 5 columnas
  • Catálogo: 5 columnas

Tablet (768px - 1024px):
  • Dashboard: 2-3 columnas
  • Users grid: 2-3 columnas
  • Catálogo: 3 columnas

Móvil (<768px):
  • Dashboard: 1 columna
  • Users grid: 1 columna (stacked)
  • Catálogo: 1-2 columnas
  • Header: Stack vertical
  • Cards historial: Flex column
```
